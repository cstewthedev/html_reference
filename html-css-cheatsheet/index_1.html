HTML Tag Reference

HTML Tags

Tag Name Description
<!DOCTYPE> Document Type The type of document you select at the beginning of a file. i.e. html, js, sql, etc.
<html> html The first parent element that's wrapped around all other elements in the html file.
<title> Title The element that will show the title on the tab of a web explorer.
<head> head The element that stores metadata from the file, linking the CSS file, linking images, etc.
<body>
<h1>,<h2><h3><h4><h5><h6> h1 A tag that signifies the size of text. h1 being biggest, h6 being smallest.
